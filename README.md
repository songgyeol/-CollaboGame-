# -CollaboGame-iOS


언제 어디서나 제약 없이 간단한 터치로 다 같이 할 수 있는 게임을 모아 미니게임으로 만들어 애플리케이션을 출시하기 위해 만들었습니다.

```swift
인물게임, 신조어게임, 초성게임, 영화명대사게임을 한 곳에 모아 다양하게 즐길 수 있는 앱을 개발함으로써,
게임에 필요한 알고리즘, 타이머, API 등 다양한 기능 구현과 활용해보는 것을 목표로 기획하였습니다.
```
